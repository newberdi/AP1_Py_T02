## Python Bootcamp #2 — Concurrency, OOP and Asynchronous Programming

A collection of Python projects focused on advanced programming concepts, including multiprocessing, asynchronous execution, object-oriented design, and file handling.

The repository contains two independent applications: an exam simulation system and an asynchronous image downloader.


### Project 1. Exam Simulation System

A simulation of an examination process involving students and multiple examiners working simultaneously.

#### Features

* shared student queue;
* concurrent examiners;
* randomized exam workflow;
* automated grading logic;
* examiner break scheduling;
* real-time monitoring of exam progress;
* statistics collection;
* final performance reporting.

#### Technologies

* Python 3
* Object-Oriented Programming
* Multiprocessing
* File Processing
* Randomized Simulation
* Console UI

#### Generated Statistics

At the end of the simulation the application provides:

* student results;
* examiner performance metrics;
* top-performing students;
* best examiners;
* students recommended for expulsion;
* most successful questions;
* overall exam success rate.

### Project 2. Asynchronous Image Downloader

A console application that downloads images from multiple URLs concurrently.

#### Features

* configurable download directory;
* directory validation;
* concurrent image downloads;
* non-blocking user input;
* network error handling;
* graceful shutdown after all downloads finish;
* summary report generation.

#### Technologies

* Python 3
* AsyncIO
* Asynchronous Tasks
* HTTP Requests
* File System Operations
