## Python Bootcamp #2 — Многопоточность, ООП и Асинхронное программирование

Набор учебных проектов, посвящённых продвинутым возможностям Python. В рамках работы были реализованы симуляция экзаменационной системы с использованием параллельных процессов и асинхронный загрузчик изображений.

Проекты демонстрируют применение объектно-ориентированного подхода, многопроцессной обработки данных, работы с файлами и асинхронного выполнения задач.


### Проект 1. Экзаменационная система

Симуляция процесса проведения экзамена между студентами и экзаменаторами.

#### Возможности

* формирование общей очереди студентов;
* одновременная работа нескольких экзаменаторов;
* моделирование экзаменационного процесса;
* случайная генерация ответов студентов и критериев оценки;
* учёт перерывов экзаменаторов;
* отображение состояния экзамена в реальном времени;
* сбор статистики по студентам и экзаменаторам;
* формирование итогового отчёта после завершения экзамена.

#### Использованные технологии

* Python 3
* Object-Oriented Programming (OOP)
* Multiprocessing
* Работа с файлами
* Генерация случайных событий
* Консольный интерфейс

#### Итоговая статистика

После завершения симуляции формируется отчёт, содержащий:

* результаты всех студентов;
* статистику по каждому экзаменатору;
* лучших студентов;
* лучших экзаменаторов;
* список студентов на отчисление;
* наиболее успешные экзаменационные вопросы;
* общую оценку успешности экзамена.

### Проект 2. Асинхронный загрузчик изображений

Консольное приложение для массовой загрузки изображений по URL-адресам.

#### Возможности

* указание каталога сохранения файлов;
* проверка доступности директории;
* параллельная загрузка нескольких изображений;
* неблокирующий ввод новых ссылок;
* обработка ошибок сети и недоступных ресурсов;
* ожидание завершения всех активных загрузок;
* формирование итоговой таблицы результатов.

#### Использованные технологии

* Python 3
* AsyncIO
* Асинхронные задачи
* Работа с HTTP-запросами
* Работа с файловой системой
